- `dashboard-expandable-row` — 可展开的供应商行
- `dashboard-expanded-row` — 展开后的授权详情行
- `dashboard-expanded-auth-row` — 展开行中的单个授权行

### 交互
- 顶部导航切换页面
- 三个计费模式区块各自独立显示
- 订阅制区块使用进度条直观展示配额使用情况
- 限流状态一览只显示被限流的授权
- **可展开行**：每个区块中的供应商行可点击展开/折叠，点击行任意位置切换
  - 默认折叠状态，行末显示展开箭头（▶）
  - 点击后展开，箭头变为 ▼，显示该供应商下所有授权的限流规则进度条
  - 一级展开不嵌套：每个授权显示其名称、Auth Key（简写）、以及该供应商所有限流规则的进度条
  - 每条规则显示为：规则标签 + 进度条（使用率百分比）+ 数值（used / max）
  - 进度条颜色规则：<80% 绿色，80-99% 黄色，>=100% 红色
  - 再次点击行可折叠

└──────────────────────────────────────┘

### 交互
- 点击 `[+ 添加模型别名]` → 弹出表单
- **策略在创建时设定，后续编辑时策略字段只读/隐藏**
- 选择供应商 → 加载该供应商的模型列表（多选复选框）
- 一个别名可关联**多个供应商的多个模型**
- 保存 → 更新 `models.yaml`

### 约束说明
- **别名必须先创建**，才能在后续流程中被选择
- **分配策略归属于别名**，与供应商无关
- 供应商和授权只负责提供模型能力和限流规则，不参与负载均衡策略决策

### 模型行字段联动规则

计费方式决定模型行显示哪些字段：

| 字段 | no_billing | per_request_weighted | per_model_token | subscription+weighted+overage | subscription+tokens+overage | subscription（超限关闭） | subscription+unlimited |
|------|-----------|---------------------|----------------|-----------------------------|----------------------------|------------------------|-----------------------|
| 模型名 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 请求倍率 | ❌ | ✅ | ❌ | ✅ | ❌ | ❌ | ❌ |
| 输入价/输出价 | ❌ | ❌ | ✅ | ❌ | ✅ | ❌ | ❌ |
| 缓存命中价 | ❌ | ❌ | 输入价有值时 | ❌ | 输入价有值时 | ❌ | ❌ |
| 建立缓存价 | ❌ | ❌ | 仅Anthropic格式 | ❌ | 仅Anthropic格式 | ❌ | ❌ |
| 别名 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

### 类型切换时的行为

| 类型 | 是否显示周期 | 可选周期 |
|------|------------|---------|
| 请求加权数 | ✅ 显示 | 5 小时 / 周 / 月 |
| Token 总量 | ✅ 显示 | 月 (固定) |
| 并发数 | ❌ 不显示 | - |

### 交互
|- 点击 `[+ 添加供应商]` → 弹出表单，表单顺序：基本信息 → 计费模型 → 模型列表 → 限流规则 → 授权 → Headers → 启用/描述
|- **计费方式放在模型列表之前**，先选计费方式，模型行自动匹配对应字段
|- **模型行字段联动**：见上方"模型行字段联动规则"表格
|- 限流规则类型切换时，周期和最大值输入框自动适配
|- **模型选择为列表下拉**，非文本输入（先加载该供应商可用模型列表）
|- 授权 Keys 集成在供应商表单内，不独立页面
|- **别名下拉选择器中列出所有已创建的模型别名**，默认选项为"不填，自动创建"（空值）
| - 别名不填（选择默认项）则自动创建：若存在与模型名同名的别名则直接关联，否则自动创建同名别名
| - 点击 `[快速添加 ▾]` → 展开下拉菜单
|   - 列表项来自 `/api/templates` 返回的模板数据（`name` 字段显示）
|   - 如果模板源为空或拉取失败 → 只显示"手动创建"和"刷新模板源"
|   - 点击模板项（如"OpenAI"）→ 弹出新建模态框，所有字段按模板预填
|     - 模板无 `models` 字段或为空则只创建默认空模型行
|     - `rate_limits` 和 `headers` 同理
|   - 点击"手动创建" → 弹出空白表单（等同于原来的 `[+ 添加供应商]`）
|   - 点击"刷新模板源" → 清缓存、强制拉取 `/api/templates?refresh=true`，成功或失败均显示 toast 提示
|- 保存 → 更新 `config.yaml`（auths 通过独立授权页管理）
|- **自定义 Headers**：JSON 格式文本输入框，用于配置自定义请求头（如 User-Agent、绕过检测头等）
|- 供应商表单不再包含授权密钥，授权管理迁移至独立的授权页面
|- **计费方式切换时**：
  - 选择`不计费` → 不显示任何计费字段；模型行只显示模型名 + 别名
  - 选择`按请求倍率计费` → 显示单价输入；模型行显示 模型名 + 请求倍率 + 别名
  - 选择`按模型 Token 计费` → 不显示额外单价字段；模型行显示 模型名 + 输入/输出价 + 缓存价 + 别名
  - 选择`订阅制` → 显示订阅价格、周期、计费方式下拉；模型行依 `billing_type` 和 `allow_overage` 而定
    - `weighted_requests` + 允许超出关闭 → 模型行只显示模型名 + 别名
    - `weighted_requests` + 允许超出开启 → 显示模型名 + 请求倍率 + 别名，并提示"请在下方模型填入超出计费的请求倍率"
    - `tokens` + 允许超出关闭 → 模型行只显示模型名 + 别名
    - `tokens` + 允许超出开启 → 显示模型名 + 输入/输出价 + 缓存价 + 别名，并提示"请在下方模型填入超出计费的 Token 价格"
    - `unlimited` → 模型行只显示模型名 + 别名
|- **模型缓存价输入**：
  - `缓存命中价` 在 `输入价` 有值且计费方式为 `per_model_token` 或 `subscription+tokens` 时显示
  - `建立缓存价` 同上条件外加 `api_format=anthropic_messages`
  - 均为可选字段，不填则按 `输入价` 计算（无折扣）

## 页面 4：授权管理 (Auths)

> 授权密钥独立管理，含新增/删除功能。**不再集成在供应商表单内**。

### 布局
```
┌─────────────────────────────────────────────────────┐
│  授权管理                         [+ 添加授权]       │
├─────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────┐   │
│  │ 供应商    │ Key        │ 名称   │ 状态     │   │
│  │ provider_a │ sk-xxx1··  │ 主key  │ 🟢 正常   │   │
│  │            │ [删除]                         │   │
│  │ provider_a │ sk-xxx2··  │ 备用key│ 🟡 接近   │   │
│  │            │ [删除]                         │   │
│  │ provider_b │ sk-yyy1··  │ 主key  │ 🔴 超限   │   │
│  │            │ [删除]                         │   │
│  └─────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

### 添加授权弹窗
```
┌──────────────────────────────────────┐
│ 添加授权                              │
├──────────────────────────────────────┤
│ 供应商: [▼ 选择供应商]                 │
│ API Key: [sk-...________________]    │
│ 名称: [如 主key_______________]      │
│                                      │
│ [取消] [保存]                         │
└──────────────────────────────────────┘
```

### 交互
- 点击 `[+ 添加授权]` → 弹出表单，从下拉列表选择目标供应商
- 供应商列表从数据库动态加载（所有已配置的供应商）
- Key 输入框明文显示，placeholder `sk-...`
| 名称可选，用于标识用途
|- 每行显示缩写 key（前 12 位 + `...`）
|- 点击 `[删除]` → 直接删除（无确认弹窗，或轻量 confirm）

### API 格式自动识别

- 供应商表单中，输入"基础 URL"时自动识别 API 格式
- 识别规则：
  - URL 包含 `anthropic` / `ikuncode` → `anthropic_messages`
  - URL 包含 `openai` / `deepseek` / `dashscope` / `stepfun` → `openai_chat`
- 识别后用户可手动切换，不限制
- 切换 API 格式时显示提示文字（示例 URL）

### 交互
- **一级 Tabs**：按授权切换，每个授权一个 tab（格式：`供应商名 / Key简称（名称）`）
- **Token 用量模块**：
  - 时间范围下拉：[24小时 ▼] [7天 ▼] [30天 ▼]
  - 折线图显示输入/输出/缓存 token 趋势
- **请求加权数模块**：
  - 固定三个数值：5小时内、本周、本月
  - 不需时间切换

## 全局交互约定

### 操作方式

| 操作 | 交互方式 |

- 🟢 正常：未触发任何限流

- 🔴 超限：触发限流规则

## 汇率与货币显示

### 全局货币选择器

- 位置：导航栏右上角
- 控件：下拉选择器，列出所有支持的货币（USD, CNY, EUR, JPY, GBP, HKD, TWD, KRW, SGD）
- 切换后自动换算所有金额并更新输入框 placeholder
- 选择偏好存储在 localStorage，页面加载时自动恢复

### 受影响区域

1. **仪表盘**：总花费卡片、各区段费用 — 使用供应商原生货币换算为用户偏好的货币
2. **供应商表单**：单价、订阅价格等输入框 label 和 placeholder 基于**该供应商的 currency**，不是全局 preferredCurrency
3. **统计面板**：所有费用数据
